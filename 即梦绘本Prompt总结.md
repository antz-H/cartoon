# 即梦绘本风生图 Prompt 总结

本文档归纳此前为「水墨绘本 / 竖版内页 / 系列延续」整理的 **Prompt 拼装方式、完整示例、负面词与留白策略**。工具以 **即梦** 为主（中文描述友好）；部分段落也可用于其他文生图模型。

---

## 一、推荐拼装顺序

单条正向提示建议按下列顺序拼接（从左到右一段接一段即可）：

1. **角色固定段**（系列必用；单张可简写）
2. **画风段**（水墨、留白、非写实等）
3. **本页情节**（本页独有的动作、场景、情绪）
4. **留白指令段**（要预留题图/短文区 **或** 满幅不留白 —— **二选一**）
5. **质量词段**（清晰度、纸感、墨色等收尾）

**说明：**

- **本页情节**：回答「画里在演什么」，每页都变。
- **质量词**：回答「清不清晰、像什么纸」，全书可共用或微调。

**若使用即梦且经常「吃掉」底部/顶部留白**：除第四节 **§4.4** 的强化段与负面 **§2.4** 外，建议把 **留白指令的最后一句**（或 §4.4 的「钉尾一句」）**整条复制到正向提示的最末尾**（质量词段之后），利用模型对首尾位置相对更敏感的特点，降低漏留白概率；仍不保证百发百中，需 **多抽几张** 或 **后期裁切** 保底。

## 二、全书通用 · 负面提示词

粘贴在即梦的 **负面提示词 / 反向提示** 栏（不要放进正向里，除非界面只有一栏）。

### 2.1 基础版（通用）

```text
照片写实，3D渲染，塑料质感，赛博朋克，日系赛璐璐厚涂，畸形手指，多余肢体，肢体粘连，三只手，毁容，血腥暴力，密集拥挤背景，强光霓虹，凌乱小字乱码，水印，Logo，低分辨率，画面糊成一团，夸张机甲，科幻城市
```

### 2.2 需要「上下留白条」时，在基础版后面追加

```text
满版无留白，四角拥挤，在画面边缘堆砌细节，底部乱草树枝，顶部复杂树冠占满，画面内嵌杂乱无章的小字
```

### 2.4 底部留白仍被画满时，在 §2.2 后再追加（专项）

即梦常把路径、山脚、草丛「延伸」进底部文字区，可再加一行负面，**专钉底缘**：

```text
底部文字区被地面画满，最下缘山石草丛路径，脚底贴底边，画面最底部复杂皴染，底边出现完整地平线占满宽边，字幕区有图案
```

（与正向里的「底部纯色纸带」同时用；仍以多抽选图为主。）

### 2.3 原创角色「墨团 & 芽仔」系列（避免与参考图角色撞车）

在基础版上可按需追加：

```text
大熊猫，龙，西方dragon，网红脸
```

## 四、留白指令段（按页选用其一，勿矛盾叠用）

### 4.1 需要预留题图 + 底部短句区

```text
版式：竖版儿童绘本内页。画面最上方留有一条明显的留白安全区（约画面高度12%），该区域仅微淡水印感纸纹或极浅渐变，不放主体人物、不承担剧情，供后期可加书名或标示；画面最下方留有一条宽阔干净的留白带（约画面高度18%），无杂物、无复杂笔触，供后期可加一两行短中文；中段为主画面叙事区。上下留白区边缘与主画区过渡柔和，避免在留白区内出现树枝、云朵、文字乱码或小道具。若不留字，整页仍像故意预留的安静空边。
```

**短版（顶 + 底）：**

```text
竖版绘本页，顶部与底部各留宽阔空白条作题图与短文区，中段为唯一叙事区，留白处无杂物无乱字。
```

### 4.2 不需要专门留白（满幅叙事 / 跨页底图）

```text
版式：竖版绘本插图采用满幅构图，主体与场景可自然延伸至画面四边，允许贴近顶边与底边；不刻意留出顶部题图带或底部文字条，背景远山云雾、地面草木可随构图需要分布至边缘。不在画面内生成书名、乱码小字或装饰性标题框。若全书后期完全不排版内嵌文字，保持纯画面叙事即可。
```

**一行版：**

```text
满幅水墨绘本竖构图，构图可顶天立地，无需顶部或底部专门留白带，画面无内嵌文字。
```

**注意：** 4.1、4.2、4.3 **每次只选一种**与当前页需求相符的写法，勿矛盾叠用。

### 4.3 底部留白供后期配文（《同望日出》示例 · 与 `作品/03 同望日出.md` 一致）

生图时**不要求模型在画内写字**：底部留出净区，中文对白**后期排版**，可避免乱码。即梦正向「留白指令段」可用：

```text
竖版内页。画面底部约20%为干净留白带，无人物无道具，仅纸色，供后期短句；其余为主画区。主画区不出现标题、对白、乱码小字、水印式杂字。
```
**短版（仅留白，无备忘）：**

```text
竖版内页，底部约20%干净留白带无人物无道具仅纸色供后期短句，其余主画区；主画区无任何内嵌文字。
```

**注意：** 若改回「由模型在画内生成固定对白」，易糊字、错字，需多抽卡并在负面词强化 `乱码，错别字，多余小字`；与 4.2「画面无内嵌文字」二选一，勿矛盾叠用。

### 4.4 即梦常漏留白时的加固（推荐）

**原因简述**：文生图模型优先「把画面画满」，对「某一条带必须空」的约束容易弱于对场景连贯性的偏好，不是提示词写错就一定会遵守。

**可叠加使用的办法（从易到难）：**

1. **负面加强**：在 **§2.2** 基础上再追加 **§2.4**（底缘专项）。
2. **正向「钉尾」**：在 **整条正向提示的最末尾**（质量词段后面）再粘贴 **同一段留白指令的最后 1～2 句**，或直接使用下面 **「钉尾一句」**；与第一节「顺序说明」中的即梦提示一致。
3. **正向「情节内占位」**（可选，与当页不冲突时用）：在 **本页情节** 段末加一句 **空间约束**，例如：`主叙事与角色重心在画面中上部至中部，地面与山脚不要延伸进入画面最下缘五分之一区域。` 避免与「脚边」「近景地面」类需求打架——若本页必须画满脚边地面，则勿叠此条，改用 1+2+4。
4. **多抽与选图**：同一套 prompt 连出多张，选底缘最干净的一张；比单张反复改词更省时间。
5. **后期保底**：成图后底部略裁或遮罩铺纸色，对白只放在设计稿里（绘本常见流程）。

**强化版 · 底部留白段（可与 4.3 二选一或叠在 4.3 后作为补充段）**

```text
【版式硬性要求】竖版绘本单页。画面最底部整宽一条约占画面高度20%至22%的区域必须为几乎纯色纸底留白：无山石、无草丛、无路径延伸、无角色肢体、无墨迹皴擦细节，仅允许极淡均匀纸肌理；该带上缘与主画面分界清晰。主叙事内容全部绘制在该留白带之上。
```

**钉尾一句（专贴正向最末尾）**

```text
强制：画面最底部约五分之一高度为干净纸色留白带，无任何景物与笔触。
```

## 五、画风段（水墨绘本 · 可每页照贴）

```text
中国水墨绘本插画，干湿笔法，大量留白，背景稀疏，可选远山淡雾或苔点草地；非摄影、非3D、非日系厚涂动漫；线条略松动、有手绘温度；高清，纸上质感。
```

（可与第三节「质量词」合并或分拆，按习惯即可。）

## 六、系列绘本 · 角色延续要点

1. **文字锁**：每页重复同一段「外貌锚点」（耳、尾、比例、配色），称呼统一。  
2. **参考图锁**：用一张满意定妆图作「角色/风格参考」，比单靠文字更稳。  
3. **组图**：同一批次改动作、少改外形词，减少漂移。  
4. **对白**：即梦内嵌长中文易糊；建议画面留白，后期统一排版字体。

## 七、原创双角色 ·「墨团 & 芽仔」固定段

**情节可换，下列角色描述建议每页保留：**

```text
系列固定主角两人：①墨团——矮胖厚重的水墨圆兽，扁圆脸细眯眼，大块墨色皴染绒毛，双耳尖各有笔锋状竖毛，粗尾下垂且尾尖有一圈更浓墨环；几乎纯水墨深浅变化，极少点缀色。②芽仔——迷你竹节身小精怪，反卷笋皮帽为头部、笋尖后翘，细肢小手足，关节呈竹节状，仅在笋帽缘与指尖有极淡新绿；体型明显小于墨团。同一绘本中二者比例与特征保持不变，儿童绘本水墨晕染、宣纸留白、竖构图、意境安静含蓄。
```

### 情节变量示例（只改【】内，接在固定段 + 画风段 + 留白段 + 质量词之后）

**初遇：**

```text
本页动作与场景：初春薄雾，墨团伏在微隆的草坡上似一块墨石，芽仔从土缝里刚探出笋帽，二者首次对视停格在一束光里，距离很近但未触碰，呼吸感，留白多，情绪柔和克制。
```

**托付：**

```text
本页动作与场景：细雨中，芽仔双手攥住墨团粗尾末端的墨环借力前行，墨团步伐慢而稳，脚边水花用淡墨点染，远方一线淡山，同行感强，不做夸张表情。
```

**驻足：**

```text
本页动作与场景：墨色老梅横斜，两三瓣浅粉点花，墨团仰头看花枝，芽仔坐在墨团肩上低头数花瓣落在掌心，一动一静，留大量空白给后期配文。
```

## 八、参考气质 · 五条完整正向 Prompt（大熊猫 + 小龙主题）

下列为**可直接整段粘贴**的正向提示（若你做**原创系列**，请改用第七节固定段，避免角色雷同）。每条末尾已含一定质量与纸感描述，可按需再接第三节质量词或第四节留白指令。

### 负面（与第二节配合使用）

同 **§2.1**，按需加 **§2.2**。

---

### Prompt ① 初遇·鼻尖相触

```text
中国水墨绘本竖构图内页，宣纸质感，墨色干湿晕染，大量留白，背景空茫薄雾与少许淡墨地面。画面主角为同一只温和的大熊猫与一条迷你水墨风小龙：大熊猫黑白分明、体态圆润、眼神安静；迷你小龙为全黑简笔细长龙形，长颈、细肢、细长须角，体型明显小于熊猫。迷你小龙立在小石上微微探身向下，大熊猫微微仰头，二者鼻尖轻轻相触，初次见面的温柔瞬间。远景可有极淡山影，道具从简，意境含蓄恬淡，情绪宁静治愈，细节干净，高清，纸上插画质感，预留上下文字留白区，非照片非3D。
```

### Prompt ② 握手·平等结盟

```text
中国水墨绘本竖构图内页，干湿墨韵，留白充足，意境安静。主角固定：圆润温和的大熊猫与迷你细长水墨小龙（全黑剪影感，长颈细肢长须角，比熊猫小很多）。二者面对面站立于浅草地与淡墨小径上，郑重而温和地伸手相握（爪与小型爪相握），平等相待的仪式感，体量对比自然不夸张。远景淡灰山峦与薄雾若隐若现，无拥挤杂物，线条柔和，宣纸肌理，高清，儿童绘本气质，非写实摄影，非动漫大眼风格，画面一侧或上下保留文字留白。
```

### Prompt ③ 同路·背负前行

```text
水墨晕染竖版绘本一页，疏朗构图，留白多，行旅意境。同一只大熊猫侧向迈步沿小路前行，头略低、步伐从容；迷你水墨小龙端正坐在大熊猫背部中央，身姿挺直，与熊猫同向望向前方。左侧近景少量竹丛与简易木栅栏一角，脚下淡墨小路，远景柔和浅灰山峦与雾气，天空留白，远处几只极简墨点飞鸟。大熊猫黑白块面边缘柔和，小龙为细长黑形长颈须角。整体宁静含蓄，墨色清润，高清，纸质插画感，非照片非3D，预留 caption 留白。
```

### Prompt ④ 看花与当下·淡彩对照

```text
水彩与淡墨结合的竖版儿童绘本内页，清爽素雅，留白充足。主角仍是同一只圆润大熊猫与同一条迷你细长水墨小龙（长颈细肢长须角）。大熊猫坐在绿草丛中，微微仰头，注视右侧深色虬曲树枝上点缀的浅粉色五瓣小花；迷你小龙坐在矮树桩上，面前铺小纸，低头专注写写画画，形成一动一静对照。天空用极浅的淡蓝水彩洗染，草地淡绿，花与枝笔触写意。整体恬静、克制、治愈，细节清楚，高清，非写实非厚重油画，非日系萌系夸张表情，画面下侧或上侧保留整齐的文字区。
```

### Prompt ⑤ 雨中同行（扩展页）

```text
中国水墨竖版绘本内页，雨意用淡墨斜线与地面水晕表现，意境温柔不压抑。大熊猫与迷你细长水墨小龙同镜：小龙坐于熊猫背上或躲在熊猫身侧，二者共撑一把简洁的竹骨小油纸伞（伞面留白或淡赭），脚下湿润泥径反光极淡，远处山影朦胧。角色外形与前几页完全一致——圆润大熊猫、全黑细长须角小龙。构图单一焦点，留白多，干湿笔法，高清，宣纸感，无现代街景水印，预留文字位置。
```

# daily-jimeng-story-prompt · 示例

下列示例演示：今日情节三块、留白文案、以及按本 skill 内 `reference/Prompt-template.md`（路径规则见 [reference.md](reference.md)）拼出的「完整即梦 Prompt」。  
角色段、画风段、留白版式、质量词与模板原文一致；仅「本页情节」随天更换。

**注意**：示例偏「示范结构」；日常生成请同时遵守 [reference.md](reference.md) **共鸣优先 · 忌任务感**——避免连续多页只换动词、不换体感与具体锚点。底部短句以 **留白对白模式** 为默认时，须写成两行 **`芽仔说：“……。”` / `墨团说：“……。”`**（弯引号）；下文示例若未带引号，仅保留情节语义，正式写入 `作品/*.md` 时应改为引号对白体。

## 示例 1 · 并肩涉水（第二种留白：只要底文）

**承接**：八日线中「涉浅溪」一类节奏；前页若已并肩，本页仍可微调为「停半步、同看水面」避免重复踏步。

**今日情节**

- **主题**：同一段凉水里，先学会同速。
- **深意**：不必一直牵手；并排、同向，已经算同行。
- **本页情节**：清浅溪流没踝，水底卵石用淡墨勾点；墨团与芽仔并排、同向立于水中，肩线大致平行，未牵手；二者同时低头看各自脚下泛开的一小圈水纹，远处仅一线淡山，构图疏朗，情绪安静克制。

**留白文案**：水纹叠在一起，又各自散开。

**完整即梦 Prompt**（段落之间空一行即可粘贴进正向提示）

```text
系列固定主角两人：①墨团——矮胖厚重的水墨圆兽，扁圆脸细眯眼，大块墨色皴染绒毛，双耳尖各有笔锋状竖毛，粗尾下垂且尾尖有一圈更浓墨环；几乎纯水墨深浅变化，极少点缀色。②芽仔——迷你竹节身小精怪，反卷笋皮帽为头部、笋尖后翘，细肢小手足，关节呈竹节状，仅在笋帽缘与指尖有极淡新绿；体型明显小于墨团。同一绘本中二者比例与特征保持不变，儿童绘本水墨晕染、宣纸留白、竖构图、意境安静含蓄。

中国水墨绘本插画，干湿笔法，大量留白，背景稀疏，可选远山淡雾或苔点草地；非摄影、非3D、非日系厚涂动漫；线条略松动、有手绘温度；高清，纸上质感。

竖版内页。清浅溪流没踝，水底卵石用淡墨勾点；墨团与芽仔并排、同向立于水中，肩线大致平行，未牵手；二者同时低头看各自脚下泛开的一小圈水纹，远处仅一线淡山，构图疏朗，情绪安静克制。

画面底部约20%为干净留白带，无人物无道具，仅纸色，供后期短句；其余为主画区。

高清，细节清楚，边缘柔和不毛糙，宣纸纸本肌理感，墨色通透不脏，印刷级绘本质感，安静不炫光。
```

## 示例 2 · 尾环借力（第一种留白：顶标 + 底文）

**承接**：若前页为「再接」类动作，句子里可与尾端墨环呼应，但避免连续多天同一抓握力度与角度。

**今日情节**

- **主题**：被需要时，身体先答应。
- **深意**：同一枚墨环，是记号，也是可借的力。
- **本页情节**：细雨或湿雾未散，墨色小径微湿；芽仔双手轻攥墨团粗尾末端那圈更浓墨环借力跟上半步，墨团未回头、肩背平稳，脚边水花用淡墨点染，远方一线淡山，同行感强，不做夸张表情与煽情对视。

**留白文案**：它把力气借在尾尖，步子就匀了。

**完整即梦 Prompt**

```text
系列固定主角两人：①墨团——矮胖厚重的水墨圆兽，扁圆脸细眯眼，大块墨色皴染绒毛，双耳尖各有笔锋状竖毛，粗尾下垂且尾尖有一圈更浓墨环；几乎纯水墨深浅变化，极少点缀色。②芽仔——迷你竹节身小精怪，反卷笋皮帽为头部、笋尖后翘，细肢小手足，关节呈竹节状，仅在笋帽缘与指尖有极淡新绿；体型明显小于墨团。同一绘本中二者比例与特征保持不变，儿童绘本水墨晕染、宣纸留白、竖构图、意境安静含蓄。
```

## 示例 3 · 第四种满幅（留白文案作画外说明）

若选用模板中「不留白区 · 满幅绘画版」，画面内仍不要生成文字；**留白文案**可留给动态、封面小字或后期单排，不塞进正向 Prompt 的画内描述。

**留白文案**（仅作画外）：风到林缘，就轻了一层。

**拼装时注意**：`[留白指令段]` 用第四种全文；`[本页情节]` 仍写清动作与环境，不要把这句中文写进「本页情节」以免模型画内出字。

# daily-jimeng-story-prompt · 参考细则

主流程见 [SKILL.md](SKILL.md)。生成 **本页情节** 与 **留白文案** 前须通读本页对应章节；**底部短句默认优先**采用下文 **留白对白模式（双行对白体）**，以保证系列语感稳定。

## Prompt 模板路径

按顺序尝试（先读到即用）：

1. `.cursor/skills/daily-jimeng-story-prompt/reference/Prompt-template.md`（本 skill 自带）
2. 仓库根目录 `reference/Prompt-template.md`（若你另行放置）

- **聊天内「完整即梦 Prompt」粘贴**：按 `即梦绘本Prompt总结.md` 推荐顺序，**角色固定段 → 画风段 → 本页情节 → 留白指令段 → 质量词**。
- **`作品/*.md` 五段节名顺序**：以 [SKILL.md](SKILL.md) 为准（留白段可在情节前，便于夹带后期排版说明）；二者语义一致，仅排版顺序不同，勿混用为两套冲突规则。

## 选题参考与约束（系列方向）

在服从连续性前提下，优先落在下列范围；视为**选题参考**与**硬约束**（除非用户当次明确改方向）。

**核心**

- 以 **「微小但重要」** 为核：停步、对视、未触到的距离、并肩无声、认错又收回、借一小点力——画面只承载 **一件具体小事**，不承载人生总结。
- 「重要」靠 **具体可辨的动作与环境** 让读者认出经验，不靠旁白下结论。

**隐性读者**

- 气质上亲近：**迷路、焦虑、忙碌** 仍愿抬头看一眼的人。
- **不要**在画内或留白里点名「献给谁」、不要贴标签或下诊断。

**留白与意义**

- 留 **问题与空隙**，不塞满答案；希望读者能想一想「为什么」。
- **严禁鸡汤**：口号式励志、命运安排式安慰、廉价治愈收尾；**`本页情节`** 避免抽象大词堆砌（如「勇敢前行」「终会光芒」）。
- 留白语气可用「轻悖论、具体物承载、两拍轻转」，见下文 **留白规则**；与鸡汤的分界以该节为准。

**生成前自问**

1. 这一页是不是 **今天路上就能发生的一件小事**？若像阶段总结，改小。
2. **`深意`** 可写在输出里备忘，**不必每页都写**；若写出来像 **阅读理解标准答案** 或 PPT 小标题，删掉。**`留白文案`** 优先像 **人话**（半句日记、略口语、可有一点笨拙），不必每页都轻悖论；仍禁鸡汤封口。
3. 允许 **悬而未决**：不必每页情绪落地为「好了」。
4. 画面里有没有 **一种具体体感或破绽**（凉、硌、渴、痒、绊、走神、笑一半憋回去、东西掉了、影子忽然很短）？若只有「安静克制、远山一线」，多半还在套层，见下节。

## 共鸣优先 · 忌任务感（防无聊）

**任务感长什么样**：像在 **完成指标**——为「微小但重要」而硬拗停步、为「不接鸡汤」而硬写聪明句、为「连续」而机械接龙；多页连用「安静克制、不做煽情对视、远景一线淡山、大量留白」**只换动词**。读者会感到 **为了什么而什么**，难以共鸣。

**共鸣从哪里来**（仍守单场景、水墨气质；不是改吵闹风）：

- **身体与感官先进场**：水温、风贴脸、晒得发烫、鞋底一滑、口干、耳鸣静下来、草叶扫过手背——**一个就够**，写进 `本页情节`，不要再用旁白解释情绪。
- **允许小不完美**：犹豫半拍、想开口又闭嘴、帮倒忙式的小动作、轻微的狼狈或尴尬——用 **一个可见动作** 呈现，忌说明书式「他感到……」。
- **留白不必句句哲学**：可以只是 **观察句**（`石头比想象中滑。`）、**口语**（`先歇口气。`）、**带一点意外**（`风把帽子掀歪了，谁也没急着扶正。`）。仍避免「终会」「只要……就……」等鸡汤封口，见 **留白规则**。
- **主题 / 深意**：聊天输出里 **可省**；若写，**一句以内**，且 **禁止**与前几页同句式排比（例如连续「××不必××」）。深意是备忘，不是本页必须完成的 KPI。

**禁止套层推进**：与上一页相比，至少动 **光 / 声 / 身体部位 / 环境一级元素** 里的一项，让读者感到 **新的具体刺激**，而不是同一机位换台词。

**快检**：把 `本页情节` 大声读一遍——如果像 **阅读理解答案** 或 **填空范文**，重写。

## 连续性规则

- 连续性优于为新奇而新奇；新奇来自推进，不来自乱加元素。
- 每页是同一条情绪线上的一步；忌夸张卖萌、吵闹、狗血（除非用户改风格）。
- 优先安静、可视叙事；**一个中心动作 + 一条清晰的情绪暗示**；保持水墨绘本气质。

**推进示例**：从注意到 → 试探靠近；从面对面 → 同向；从距离的紧 → 认出；从景 → 同频；从张力 → 再接。

**避免**：相邻天重复握手或同pose；杂乱道具；破坏比例与设定；日系厚涂海报、3D、摄影写实风。

## 留白规则

有底部/顶部留白指令时，写 **留白文案**（后期排版，勿让模型画长文）。

**须满足**

- 与画面内容直接相关；平静、可读。
- 禁鸡汤口号、说教、硬抬升；禁 **封口式**道德总结（整页意义一句话钉死）；允许 **轻声改框、留一问、留半拍**。
- **篇幅**：以一句为主；需要时 **两句极短** 同一拍，排版两行。

**质感**：轻、具体、克制、略可回味但不沉重。

### 参考语感（本系列）

按当页改写，勿套抄；**不必每页都用悖论**，口语与观察句同样成立。

- **轻悖论 / 改框**：例：`不知道去哪里，并不意味着你迷失了方向。`
- **具体物承载**：例：`那棵树经历了不少磨难，可它还在那儿，还收获了力量和美丽。`
- **抑扬两拍**：例：`什么事情都没有发生。` / `也许，是先发生在地面之下。`
- **口语 / 小笨拙**（增）：例：`先站一会儿。` / `鞋带散了，弯腰的时候影子叠在一起。`

**与鸡汤分界（仍禁止）**：「终会」「只要……就……」、命运安排、治愈一切式收尾。允许 **说轻、判断留给读者**。

**极简白描仍可用**：`雨细下来，路就慢了一点。` / `它们没有说话，只把脚步放在了一起。`

**反面**：`只要一起前行，终会迎来属于自己的光。` 等。

### 留白对白模式（双行对白体 · 推荐默认）

本系列底部【后期排版 · 短句】**优先**写为 **两行、两人、各一行**，且须呈现为 **「说出来的话」**：每行固定为 **`某某说：“台词。”`**——先写角色与 **说**，接 **中文弯双引号** **`“`** **`”`** 包裹台词（与纸书正文对话排版一致），句内可用逗号，句末用句号、问号或省略号为宜，避免一串叹号。语气像 **短童话对白** 或 **极轻寓言**，**不是**旁白解说。除非当页确实无对话口（纯景语），再退回到上文「观察句 / 口语白描」两行（退避时仍可为无引号白描两句）。

**排版示例（勿照抄内容）**

```text
芽仔说：“脚步再快点吧，天要斜了。”
墨团说：“急什么，烫还没散完呢。”
```

**稳定「钩子」（结构模式，勿套抄原句）**

下列为 **句式骨架**，每页按当页画面与连续性 **重写内容**，避免连续多页同一钩子类型。

| 钩子类型 | 读者效果 | 结构提示（示意） |
|----------|----------|------------------|
| **急 × 停** | 一脚油门一脚刹车，中间空半拍 | 一方催时间/赶路/怕晚 → 另一方 **放慢动作** 或 **把理由换一面说**（不是吵架，是节奏错开） |
| **问 × 极简答** | 像「你在想什么 / 什么都没想 / 真好」 | 一方短问 → 另一方 **答得更短或答非所问**，答里 **留白**，不把道理说满 |
| **绕远 × 值得** | 走错路才遇见花园 | 一方嘀咕远/错/绕 → 另一方点头或一句 **把「远」轻轻改框**（仍须贴合本页具体物/动作，忌空泛） |
| **怕不被喜欢 × 自路** | 边界感，略硬但落在对话里 | 一方暴露担心 → 另一方 **一句立场**；避免排比口号，**禁止**「终会」「只要…就…」式封口 |
| **季候 × 约定** | 时间往前推半步，温柔不甜腻 | 一方点季节/将变天 → 另一方 **落到具体共享小事**（如一起做什么），忌宏大承诺体 |
| **点头沉默** | 动作替台词 | 一行可写 **`墨团点点头：“……。”`**（动作 + 说 + 引号台词同列一行）；另一行仍是 **`芽仔说：“……。”`**；总仍 **两行** |

**写法硬规（与留白规则并存）**

1. **两行各一句对白、格式统一**：每行均为 **`芽仔说：“……。”`** 或 **`墨团说：“……。”`**（或 **`墨团点点头：“……。”`** 等极简动作 + `说` + 引号台词）；**严禁**省略双引号、**严禁**把两行合成一段无 `说` 字的「金句旁白」。总字数仍控制在 **后期两行内可读**，勿写成长段落。
2. **与 `本页情节` 同拍**：对白里的意象（烫、风、脚步、天光等）须能在画面上找到落点；**禁止**文案与画各说各话。
3. **改框句**：允许像「迟到 / 其实有人在等」这类 **轻声改框**，但必须 **具体、短**，且 **不接** 鸡汤甜尾；判断标准是：读完是否仍像 **两个人在走路时随口说出**，而非作者跳出来总结。
4. **轮换**：与上一页相比，钩子类型 **尽量换一类**（避免连续三天都是「急×停」）；若上一页已是对白体，本页 **仍用对白体**，但 **换句式内部节奏**。
5. **角色名**：固定 **芽仔**、**墨团**（与本系列设定一致）；不要用用户举例里的其他昵称，除非用户当次指定。

**生成前快检（对白版）**

- 每行是否同时具备 **`说` + 弯引号 `“` `”` 包裹的台词**？主画区正向 prompt 仍 **禁止**画内生成文字；引号体仅用于 **后期落在底部留白** 的排版文案。
- 去掉角色名与 `说`、只读引号内台词，两行是否仍是 **两个活生生的人在顶嘴或接话**，而不是 **美文摘抄**？
- 第二行是否 **留有空隙**（读者能停一下），而不是 **把道理钉死**？
- 大声读一遍：有没有 **教案感 / 演讲感**？有则缩短、改口语、或改用「问×极简答」。

## `本页情节` 写作标准

- 单一场景；动作与环境同写；**情绪靠动作与感官托住**，不用「感到、明白、领悟」类词。
- 简练、可嵌入正向 prompt；**画面可执行**；至少含 **一个可画的具体锚点**（物件、光影、身体细节、声音来源之一）。
- 风格：克制、略诗意不堆砌、图像优先；**忌把「安静克制」当万能尾句重复用**。

## 默认输出结构（聊天）

除非用户另有要求：

```markdown
今日情节

主题：...（可选；忌教案标题）
深意：...（可选；忌与前几页同句式；可写「无，画面自证」）
本页情节：
...
留白文案：（默认两行对白，弯引号）
芽仔说：“……。”
墨团说：“……。”

完整即梦 Prompt
...

连续性说明
- 承接：...
- 变化：...
- 原因：...

**已写入**：`作品/{两位序号} {标题}.md`
```

若用户只要 prompt：内部仍做连续性推理，输出 **`本页情节`、`留白文案`、`完整即梦 Prompt`**。

## 输入不全时

- 连续性不清：从最近文件推断位置 → 简短说明假设 → 给保守、易改的一拍。
- 模板缺失：向用户索要，或说明后沿用最近一次已知结构。

## 边界

本 skill 止于 **prompt 与作品 md**。除非用户另请且工具有效，勿声称已出图、已选图、已修图。

## 即梦易漏留白时（工具侧）

模型不保证严格留白带。生成时可嘱用户：见仓库 **`即梦绘本Prompt总结.md` §4.4**（强化留白段、钉尾一句、情节内占位可选）与 **§2.4** 负面追加；正向 **留白句钉在全文最末** 与 **多抽选图** 往往比单段描写更有效。

写入 `作品/*.md` 时，在 **`[质量词：高清、宣纸质感]`** 之后**固定追加** `[即梦钉尾 · 正向全文最末粘贴]`（文案同 §4.4「钉尾一句」），便于复制进即梦时钉在正向最末。

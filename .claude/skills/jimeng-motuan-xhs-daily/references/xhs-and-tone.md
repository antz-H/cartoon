# 小红书与人感约束（本 skill 摘要）

与仓库内 `daily-jimeng-story-prompt` 的 reference 可对照阅读；本文件只保留本 workflow 必需项。

## 小红书主标题

- **3～10 个汉字**，是对**当页主题/画面一拍**的总结，读者一眼能懂。
- 用**完整短语**，禁为生造「雅称」硬凑两字复合词；禁模板腔（如机械「两字｜副标」）。
- **不**拼进即梦正向 prompt。

### 落盘前标题二检（agent 自用）

1. 去掉书名号/引号后，是否仍是**白话名词或短动作**（而非抽象格言）？
2. 与近 2～3 页标题并列看：**是否换了一种句式或意象**，避免缩略图级「同一句式换词」？不过线则改标题再写入 `## 小红书主标题`。

## 对白方言

- 留白对白默认使用**方言**，增加趣味性与体感。方言不限地域——四川话、东北话、粤语、闽南话、湖南话、山东话……只要**有特色、有画面感**即可；每拍根据场景情绪选择最贴的方言，不必全篇同一方言。
- 选方言的原则：哪句方言说出来比普通话**多出体感或画面**，就用哪句。方言不是逐字翻译，是用方言重新说——同一句话用方言说出来要有**新发现**（更具体、更有肉感、更像真人在路上接话）。
- 方言选择动词、体感词优先（四川"拱出来"优于"长出来"、"脚杆"优于"脚"；东北"秃噜"优于"滑"；粤语"企"优于"站"；湖南"晓得"优于"知道"）。
- 芽仔方言偏急、偏小孩气（短句、叠词、反问）；墨团方言偏稳、偏按节奏（"莫忙""再讲""歇着"）。
- 对白释义中需点明方言词比普通话多出了什么体感/画面，不只是翻译。
- 发布包装（正文首行、唯一句、互动问法）可适度带方言味，但不过度——让普通话读者也能读懂。

## 发给读者的正文气质

- **共鸣优先**：具体小破绽、体感、关系张力；禁空泛「安静克制」当内容。
- **禁鸡汤、禁教案感、禁 AI 腔**：少用排比、少用「让我们」「其实人生」类套话；像真人在路上接话。
- **留白对白**（若本 run 需要两行配文）：腹稿先用口语，再套 `芽仔说："……。"` / `墨团说："……。"`（中文弯引号）；去掉「某某说」后仍要像两人在顶嘴/接话，**不像**签文对仗或格言填空。

## 可选附录（用户若要发笔记）

- **对白释义**：一小段，解释两句在画面上指什么；**不**重复整张画的第二套旁白，**不**与标题全文重复。
- **发布包装**：首行钩子、互动问法、封面/缩略图自检（顶标区、亮度、避免同构图只换台词等）——按需展开，保持简短。

## 信息流自检（极简）

相对近 2～3 页，至少换一样：**天色 / 机位 / 主道具 / 身体关系**之一，避免缩略图级同构。

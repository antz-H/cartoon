# 小红书与人感约束（本 skill 摘要）

与仓库内 `daily-jimeng-story-prompt` 的 reference 可对照阅读；本文件只保留本 workflow 必需项。

## 小红书主标题

- **3～10 个汉字**，是对**当页主题/画面一拍**的总结，读者一眼能懂。
- 用**完整短语**，禁为生造「雅称」硬凑两字复合词；禁模板腔（如机械「两字｜副标」）。
- **不**拼进即梦正向 prompt。

## 发给读者的正文气质

- **共鸣优先**：具体小破绽、体感、关系张力；禁空泛「安静克制」当内容。
- **禁鸡汤、禁教案感、禁 AI 腔**：少用排比、少用「让我们」「其实人生」类套话；像真人在路上接话。
- **留白对白**（若本 run 需要两行配文）：腹稿先用口语，再套 `芽仔说：“……。”` / `墨团说：“……。”`（中文弯引号）；去掉「某某说」后仍要像两人在顶嘴/接话，**不像**签文对仗或格言填空。

## 可选附录（用户若要发笔记）

- **对白释义**：一小段，解释两句在画面上指什么；**不**重复整张画的第二套旁白，**不**与标题全文重复。
- **发布包装**：首行钩子、互动问法、封面/缩略图自检（顶标区、亮度、避免同构图只换台词等）——按需展开，保持简短。

## 信息流自检（极简）

相对近 2～3 页，至少换一样：**天色 / 机位 / 主道具 / 身体关系**之一，避免缩略图级同构。

# 落盘前自检 · 反思闭环

不自评艺术好坏，只卡协议与可执行性。不过线时**静默修订一轮**再交付（只改文字不重起主题）；仍卡壳→聊天说明，禁止谎称已写入。

---

## A · 即梦正向 Prompt

| 检查项 | 过线标准 | 不过线时 |
| --- | --- | --- |
| 五段顺序 | 角色→画风→留白（四选一）→本页情节（**以`竖版内页。`起笔**）→质量词 | 重排；勿把画风段重复塞进本页情节 |
| 留白段 | 与 [Prompt-template.md](Prompt-template.md) 所选版本原文一致 | 对照模板改回 |
| 可画性 | 单场景、动作与机位一句能说清 | 删抽象形容词，补可见动作 |
| 与 memory | 无直接矛盾 | 以 memory 为准改 prompt，选型理由半句说明取舍 |
| 与近 2～3 页 | 天色/机位/道具/身体关系至少变一项 | 改本页情节或选型，选型理由写明改了哪项 |

---

## B · `作品/{两位序号} {标题}.md`

| 检查项 | 过线标准 |
| --- | --- |
| 路径 | 新序号 = 现有最大 +1；不得覆盖已有 |
| 三节齐全 | 主题与选型理由 → 小红书主标题 → 即梦正向 Prompt 顺序正确 |
| 主标题 | 一行 3～10 汉字，不像模板腔 |
| Prompt 节 | 代码围栏内完整可复制；钉尾含于本节 |
| 诚实性 | 落盘失败则聊天不得写「已写入」 |

---

## C · `memory.md`（仅用户明确「是」之后）

执行 [memory-protocol.md](memory-protocol.md) 前，额外确认：

1. 已读旧版并 A/B/C 槽位化再合并
2. 全文件 ≤300 字
3. 禁止本轮剧情复述占满 B/C；禁止贴整段 prompt
4. 用户未确认→绝不写
5. 正文主体为 C/B 可迁移短句；无序号/路径/数字作独立信息
6. 已通过具象扫描：删「离开本篇就不成立」的句子或升格为维度级规则
